# ffmpeg-android-directory
This is the missing directory to include in the ffmpeg folder within addons

## steps to fix android directory missing
1. Download this zip folder
2. Extract the folders
3. copy the "android" folder into the ffmpeg folder within "addons"
  
NOTE:You must already have the latest ffmpeg addons from here: https://github.com/EIRTeam/EIRTeam.FFmpeg/releases/
